Features

Everything a frontend dev needs

INTERCEPTION

Full fetch + XHR capture

Patches window.fetch and XMLHttpRequest at document_start — before any page script executes. Captures method, URL, headers, request body, response body, status, and timing.

SMART ROUTING

Auto path templating

Numeric IDs, UUIDs, and hashes in URL segments are automatically replaced with named parameters. /users/42 and /users/99 merge into /users/{userId}.

SCHEMA

Deep schema inference

Infers JSON Schema draft-7 from real response bodies. Handles nested objects, arrays, nullable fields, and detects formats — date-time, email, UUID, URI.

MOCK SERVER

Realistic faker fallback

If exact response data isn't captured, the server generates realistic fake data from the inferred schema using @faker-js/faker — emails, names, dates, UUIDs.

FOSS

Zero proprietary dependencies

The Chrome extension has zero npm runtime dependencies. The server uses only MIT-licensed packages. No cloud APIs, no accounts, no telemetry. Your data stays local.

DASHBOARD

Local web dashboard

The mock server ships with a web dashboard at /dashboard — inspect all registered routes, view captured response data, and import new specs without restarting.
